Sortiranje

Bubble sort

Bubble sort prolazi kroz niz i menja pozicije susednih elemenata ukoliko oni nisu sortirani.

Ovaj postupak se ponavlja sve dok čitav niz ne bude sortiran, odnosno, izvršavanje se prekida ukoliko u jednom prolazu nije došlo ni do jedne izmene.

Vremenska složenost: n^2
Prostorna složenost: 1 
void bubble_sort(int *array, int n) {
    while (1) {
        int cnt = 0;

        for (int i = 0; i < n-1; i++) {
            if (array[i] > array[i+1]) {
                int tmp = array[i];
                array[i] = array[i+1];
                array[i+1] = tmp;
                cnt++;
            }
        }

        if (cnt == 0) break;
    }
}

Shaker sort

Shaker sort funkcioniše na sličan način kao bubble sort.

Shaker sort i bubble sort razlikuju se po tome što shaker sort naizmenično prolazi kroz niz od početka ka kraju, odnosno od kraja ka početku.

Vremenska složenost: n^2
Prostorna složenost: 1 
void shaker_sort(int *array, int n) {
    int turn = 1;
    while (1) {
        int cnt = 0;

        if (turn) {
            for (int i = 0; i < n-1; i++) {
                if (array[i] > array[i+1]) {
                    int tmp = array[i];
                    array[i] = array[i+1];
                    array[i+1] = tmp;
                    cnt++;
                }
            }
            turn = 0;
        } else {
            for (int i = n-2; i >= 0; i--) {
                if (array[i] > array[i+1]) {
                    int tmp = array[i];
                    array[i] = array[i+1];
                    array[i+1] = tmp;
                    cnt++;
                }
            }
            turn = 1;
        }

        if (cnt == 0) break;
    }
}

Selection sort

Selection sort pri svakom prolasku bira najmanji element iz nesortiranog dela niza i menja mu poziciju sa prvim nesortiranim elementom.

Ovaj postupak se ponavlja n-1 puta, odnosno dok čitav niz ne postane sortiran.

Vremenska složenost: n^2
Prostorna složenost: 1 
void selection_sort(int *array, int n) {
    for (int i = 0; i < n-1; i++) {
        int minIndex = i;
        for (int j = i+1; j < n; j++) if (array[j] < array[minIndex]) minIndex = j;

        int tmp = array[i];
        array[i] = array[minIndex];
        array[minIndex] = tmp;
    }
}

Insertion sort

Insertion sort prolazi kroz niz i svaki element stavlja na odgovarajuću poziciju u sortiranom delu niza.

Vremenska složenost: n^2
Prostorna složenost: 1 
void insertion_sort(int *array, int n) {
    for (int i = 1; i < n; i++) {
        int key = array[i];
        int j = i-1;
        for (; j >= 0 && array[j] > key; j--) {
            array[j+1] = array[j];
        }
        array[j+1] = key;
    }
}

Merge sort

Merge sort rekurzivno deli niz po sredini na dva jednaka dela, sve dok ti delovi ne postanu jedinične veličine.

Merge sort zatim prolazi kroz sortirane polovine i spaja ih u novi sortirani niz.

Spajanje se vrši poređenjem tekućih elemenata sortiranih polovina. Manji tekući element se smešta na tekuću poziciju novog niza, a zatim se inkrementira brojač niza iz kog manji element potiče kao i brojač novog niza.

Preostali elementi jednog od sortiranih nizova se na kraju kopiraju na kraj novog niza.

Vremenska složenost: n*log(n)
Prostorna složenost: n 
void merge_sort(int *array, int low, int high) {
    if (low < high) {
        int mid = (low + high) / 2;

        merge_sort(array, low, mid);
        merge_sort(array, mid+1, high);

        int n1 = mid - low + 1;
        int n2 = high - mid;

        int tmp1[n1];
        for (int i = 0; i < n1; i++) tmp1[i] = array[i+low];

        int tmp2[n2];
        for (int i = 0; i < n2; i++) tmp2[i] = array[i+mid+1];

        int i = 0, j = 0;
        while (i < n1 && j < n2) {
            if (tmp1[i] <= tmp2[j]) array[low++] = tmp1[i++];
            else array[low++] = tmp2[j++];
        }

        while (i < n1) array[low++] = tmp1[i++];
        while (j < n2) array[low++] = tmp2[j++];
    }
}
